# 🤖 AI Code Assistant Agent

An AI-powered debugging assistant that analyzes, fixes, and executes Python code using Large Language Models (LLMs).

This tool helps developers detect errors, understand issues, and automatically generate corrected Python code.

---

## 🚀 Features

* 🔍 AI-powered code debugging
* 📊 Static code analysis using **Pylint**
* ▶ Run Python code and view execution output
* 🛠 Extract corrected code from AI responses
* ⬇ Download corrected Python code
* 📂 Upload `.py` files for analysis
* 🗑 Clear code editor
* 📜 Analysis history sidebar
* 🧠 AI explanation of errors
* 💻 Clean interactive **Streamlit UI**

---

## ⚙️ Tech Stack
